Chatra — FastAPI backend with RAG

Chatra is an educational platform where students join classes, submit assignments, and track progress. I'm building the backend that powers it: a FastAPI service designed for clarity and reliability, with a retrieval-augmented pipeline in the works.

  • FastAPI REST API with JWT authentication
  • PostgreSQL data models for classes, students, and assignments
  • Dockerized for consistent, repeatable deployment
  • RAG pipeline in progress — to make course content searchable and queryable by an LLM

RugPull
Hackathon MVP · Crypto Security

RugPull — AI token risk scanner

Built in 48 hours at the Cybershield 2026 hackathon. RugPull scans crypto tokens and wallets for fraud signals — smart contract risks, onchain activity, social mentions, and whale concentration — and returns an AI-generated verdict. Pitched to industry judges, placed Top 5.

  • Smart contract analysis — mint/blacklist functions, owner privileges, verification status
  • Onchain analysis — transaction activity, liquidity, holder distribution
  • AI verdict via GPT-4o — a 0–100 risk score with a written explanation
  • AML wallet checks — transaction history screened against mixers and known scams
